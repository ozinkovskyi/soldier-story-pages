# Jingle Bellz

Landing page for the book *Jingle Bellz* by Oleksii Anulia.

## Tech stack

- Vite
- TypeScript
- React
- shadcn/ui
- Tailwind CSS
- i18next (en, ru, ua, de, nl)

## Run locally

```sh
npm i
npm run dev
```

## Build

```sh
npm run build
```

Static output is in `dist/`. Deploy `dist/` to any static host or use the included GitHub Actions workflow for GitHub Pages.
